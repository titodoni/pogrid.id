# Roles and Access

Three actor types use POgrid.id. Each has a distinct surface.

- **Owner** — runs the business. Sees the Matrix Dashboard, KPIs, Client Scoreboard, PO/Item directory, user management.
- **Admin** — back-office. Creates/edits POs and items, picks which stages an item gets (no auto-injection — see [[Core Paradigm]]).
- **Worker** — shop floor. Logs progress from their phone via PIN login; sees only their assigned stages.
