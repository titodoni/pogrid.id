# Workflow Stages

The production pipeline as it appears on the Owner Matrix Dashboard. Items flow left → right; stages are independent under the [[Core Paradigm]].

1. **Drafter** — drawing / design
2. **Purchasing** — buyout / material procurement
3. **Productions** — CNC / Milling / Welder / Helper (concurrent roles)
4. **QC** — quality control
5. **Finance** — invoicing / payment
6. **Delivery** — out the door

## Lifecycle status mapping
Raw DB enums are localized for display:
- `MANUFACTURE` → `Produksi Internal / Manufactured`
- `BUYOUT` → `Beli Jadi / Buyout`
- `APPROVED` → `Gambar Disetujui / Drawing Approved`
