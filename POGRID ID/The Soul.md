# The Soul

> One question defines POgrid.id: *"Where is my order right now, and will it be on time?"*

POgrid.id is a **Live Progress & Delivery Punctuality Tracker** for Indonesian SME manufacturing workshops (CNC shops, fabrication houses, full engineering workshops).

It is **NOT**:
- an ERP
- an MES
- an inventory / raw-material / warehouse system
- an accounting tool

No Bill of Materials. No stock tracking. No warehouse management. Just real-time progress and punctuality.

It translates chaotic shop-floor reality into clean, undeniable data — visible on the Owner's dashboard and each worker's phone in seconds.
